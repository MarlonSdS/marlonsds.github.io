Linguagens, tecnologias e frameworks

Aqui estão estão listadas as linguagens e tecnologias que conheço e também alguns frameworks que já utilizei com cada uma delas. Para ver os frameworks basta clicar sobre o texto de alguma linguagem/tecnologia para expandir seus detalhes.

Javascript
Eu não teria criado esse site do jeito que ele é sem Javacript, acho que não há muito o que dizer, é uma linguagem bem básica que todo desenvolvedor web conhece. E é também a base para boa parte das stacks listadas aqui.

NodeJs
Uma das principais frameworks de desenvolvimento backend com js. Fiz um curso de Node na Udemy enquanto ainda estava na faculdade e desde então usei em alguns projetos.

Vue
Vue, outra ferramente para frontend com javascript e typescript. É basatante eficiente e fácil de se trabalhar.

React
React, uma excelente biblioteca para frontend com javascript. Também fiz um curso na udemy enquanto estava na faculdade. É bastante prático embora fique um pouco caótico em projtos grandes.

Python
Linguagem que estou estudando atualmente e pretendo me aprofundar. É muito boa para fazer automação de certas tarefas e, pelo que eu vi, é bastante usada no mercado.

MySql
Creio que todo desenvolvedor e analista que se preze conhece a MySql, é a linguagem de bd mais usada e a mais básica. Eu tive contato com ela a primeira vez na faculdade e bem, a usei em praticamente todos os meus projetos desde então, não há muito o que dizer.

PHP
Primeira linguagem web que tive contato e a que mais usei na prática já que naquele projeto do Ineves o php com o framework code igniter era a linguagem mais usada.

Java
Primeira linguagem de programação que tive contato e fiz alguns trabalhos amadores. Apesar de achar uma linguagem muito "gostosinha" de se trabalhar nunca me aprofundei muito nela mas posso me aprofundar se um dia aparecer nessecidade.
